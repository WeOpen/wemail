# WeMail Logo Design

**Date:** 2026-04-14

## Goal

为 `WeMail` 设计一个极简科技感项目 logo：主体是一封信，中央折痕与字母 `W` 融为一体。

## Decisions

- 采用单色几何风格，优先服务小尺寸识别和顶部品牌位展示。
- `W` 不额外叠加字形，而是直接由信封上沿折痕演化出来。
- 先实现为可复用的内联 SVG 组件，方便在 React 界面中继承颜色和尺寸。
- 首次接入放在工作台顶栏品牌位，替换当前通用邮件图标。

## Visual Notes

- 外轮廓为利落的信封矩形，少圆角或不依赖圆角。
- 上沿折线形成 `W`：左上角进入第一谷底，上抬到中峰，再下探第二谷底，最后回到右上角。
- 整体靠轮廓和折线识别，不依赖渐变、阴影或多色。

## Verification

- 补一个顶栏品牌位回归断言，确认新 logo 被渲染。
- 运行 web 侧测试、typecheck、lint、build。
