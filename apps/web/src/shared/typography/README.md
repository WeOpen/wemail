# ✍️ shared/typography

共享排版原语层。

## ✅ 放什么
- `Heading`
- `Text`
- `Label`
- `Muted`
- `Code`
- `Kbd`
- 基于语义字号、tone 和原生标签的轻量排版封装

## 🚫 不放什么
- 页面级富文本排版系统
- Markdown 渲染器
- 业务专属文案拼装逻辑

## 使用约定
- 标题优先用 `Heading`，正文优先用 `Text`
- 弱化说明、时间戳、辅助信息用 `Muted`
- 字段标题或可点击表单标签用 `Label`
- 内联标识符、环境变量、命令片段用 `Code`
- 快捷键提示统一用 `Kbd`
