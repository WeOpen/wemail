# Code of Conduct

## Our Pledge

我们希望 `wemail` 成为一个专业、尊重、可协作的项目。

所有参与者都应：
- 尊重他人
- 聚焦问题本身
- 提供建设性反馈
- 避免人身攻击、羞辱或敌意表达

## Expected Behavior

- 用事实和证据讨论问题
- 在 review 中指出问题时给出改进方向
- 尊重项目边界、流程和维护者决定

## Unacceptable Behavior

- 人身攻击
- 歧视性或侮辱性语言
- 恶意刷屏、骚扰
- 故意破坏协作流程

## Enforcement

维护者有权删除不当内容、拒绝恶意贡献、限制破坏性参与。
